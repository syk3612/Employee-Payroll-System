Employee Payroll System is a PL/SQL-based application developed using Oracle Database to manage employee information, attendance tracking, payroll processing, 
and salary management.The project demonstrates core database development concepts such as table design, primary and foreign key relationships, 
PL/SQL packages, procedures, exception handling, and centralized error logging.
The system maintains employee and department data, records attendance details, generates salary information, 
and stores payroll history for future reference. A centralized error logging mechanism has been implemented to capture and track application errors, 
improving maintainability and debugging.
This project is designed to simulate a real-world payroll management solution while showcasing practical Oracle SQL and PL/SQL development skills.
